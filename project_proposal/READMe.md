# Project Proposal

## Project Title
Design and Simulation of a Virtual Enterprise Network Using Cisco Packet Tracer and Wireshark

## Project Description
This project focuses on designing and simulating a virtual enterprise network using Cisco Packet Tracer and Wireshark. The goal is to create a functional network environment that demonstrates important computer networking concepts.

## Project Objectives

- Design a virtual enterprise network topology
- Configure routers, switches, servers, and client devices
- Implement DHCP and DNS services
- Test communication between network devices
- Analyze network traffic using Wireshark
- Troubleshoot and verify network performance
