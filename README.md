# Introduction to Computer Networks Project

## Project Title
Design and Simulation of a Virtual Enterprise Network Using Cisco Packet Tracer and Wireshark

## Project Overview
This project focuses on designing and simulating a virtual enterprise network using Cisco Packet Tracer and Wireshark. The project demonstrates network design, device configuration, communication between devices, and network traffic analysis.

## Project Objectives

- Design a virtual enterprise network topology
- Configure routers, switches, servers, and client devices
- Implement DHCP and DNS services
- Test network connectivity
- Analyze network traffic using Wireshark
- Troubleshoot and verify network performance

## Tools Used

- Cisco Packet Tracer
- Wireshark
- GitHub

## Network Services

- IP Addressing
- DHCP
- DNS
- Routing
- LAN Communication
- Packet Analysis
